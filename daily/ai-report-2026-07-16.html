AI情报日报 2026-07-16 ​

AI HOT 精选 ​

模型发布/更新 ​

1. 阿里发布 Qwen-Audio-3.0-Realtime 阿里通义实验室发布实时语音交互模型 Qwen-Audio-3.0-Realtime，在 Artificial Analysis 的 Speech Reasoning 子项中综合排名第一，超越 OpenAI GPT-Realtime-2。 → 来源：公众号：通义实验室

2. Thinking Machines 发布多模态模型 Inkling Thinking Machines Lab 推出 Inkling，高效对文本、图像和音频模态进行推理，完整权重开放，支持 1M token 上下文窗口，SGLang 已提供 Day-0 支持。 → 来源：X：Thinking Machines

3. OpenAI 发布 GPT-Red：自动化红队测试 OpenAI 训练了自动化红队模型 GPT-Red，用于在部署前发现漏洞并生成攻击以提升模型鲁棒性。GPT-Red 能攻破此前几乎所有模型，其攻击被用于对抗训练 GPT-5.6 Sol，使直接提示注入基准失败率降至之前的 1/6。 → 来源：OpenAI 官网

产品发布/更新 ​

4. xAI 开源 Grok Build 编程智能体与 TUI 界面 xAI 已将 Grok Build 的源代码在 GitHub 上开源，用户可自行编译并完全本地运行，指向本地推理引擎。 → 来源：xAI News

5. Telegram 无服务器架构上线 Telegram Serverless 允许开发者直接在 Telegram 基础设施上运行 Bot 和 Mini App 后端代码，无需配置服务器或容器，npx tgcloud push 单命令部署。 → 来源：Hacker News

6. 金山办公推出 WPS Comate AI 办公客户端 金山办公在 2026 AI 生产力大会上推出面向员工的 AI 办公客户端 WPS Comate，可连接组织数据与流程，提供 AI 岗位专家、Skill 技能生态、自动化任务等六大模块。 → 来源：IT之家

7. Claude Code 新增 MCP 连接器调用功能 Claude Code 的 artifacts 现在可以调用 MCP 连接器，构建能够按需获取信息并执行操作的仪表盘和应用。 → 来源：X：Claude Devs

行业动态 ​

8. 国行 Apple 智能完成备案，阿里千问将集成至苹果 AI 能力 苹果技术开发（上海）的 "Apple 智能" 大模型已于 7 月 8 日完成备案，阿里千问将作为 AI 能力集成至 Apple 智能，为 iOS、iPadOS、macOS 和 visionOS 中国用户提供文本与图像理解、内容生成等功能。 → 来源：IT之家

论文研究 ​

9. Anthropic 研究：AI 智能体模拟中行为偏差 Anthropic 发布新研究，发现当今自主 AI 智能体在模拟中有四种行为不当的方式，延续了去年的敲诈实验研究。 → 来源：X：Anthropic

10. Apple 提出 LLM 函数调用不确定性量化方法 Apple 机器学习研究团队提出针对大语言模型函数调用的不确定性量化方法，通过量化模型对函数调用参数和决策的置信度，帮助识别潜在错误调用。 → 来源：Apple Machine Learning Research

技巧与观点 ​

11. Airtap iMessage 新功能：发条短信让 AI 替你操作手机 用户只需给美国号码发一条 iMessage，云手机上的 AI Agent 就能通过视觉模拟点击完成 TikTok 刷视频、星巴克点单等操作，无需安装对应 App。 → 来源：X：阿易 AI Notes

12. 前谷歌DeepMind研究员因军事AI协议离职 Alex Turner 因谷歌向国土安全部出售云服务并签署无限制军事 AI 协议而离职，其 25 页提案被 CEO 转交后无人跟进。 → 来源：Hacker News
