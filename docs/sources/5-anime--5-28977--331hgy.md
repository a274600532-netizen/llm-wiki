# Gintama° — MyAnimeList Entry

《Gintama°》（银魂°）是《银魂》系列的第四季动画，于2015年首播，共51集。该作品在MyAnimeList上获得9.05的高评分，类型涵盖动作、喜剧与科幻。

## 剧情概要

坂田银时、志村新八和神乐三人组成的万事屋团队回归。故事发生在一个被外星人（天人）征服、禁刀的架空江户时代，万事屋三人组靠接各种零活维生。然而新八和神乐仍然没有拿到工资——银酱真的把所有钱都花在玩小钢珠上了吗？

某夜，银时醉醺醺地回家时，一艘外星飞船坠落在附近。一名重伤的外星人……
