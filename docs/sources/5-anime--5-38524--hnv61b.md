# Shingeki no Kyojin Season 3 Part 2 — MyAnimeList 条目

- **发布日期**: 2026年07月10日
《Shingeki no Kyojin Season 3 Part 2》（进击的巨人 第三季 Part 2）是2019年播出的日本动画，改编自谏山创的同名漫画。该季在MyAnimeList上获得9.05的高评分，共10集，状态为"Finished Airing"，类型涵盖动作、剧情与悬疑。

## 剧情概要

为了恢复人类日益渺茫的希望，[survey-corps](/entities/survey-corps-调查兵团)发起了一场夺回[wall-maria](/entities/wall-maria-玛利亚之墙)的任务，与无情的"巨人"（Titans）的战斗再次拉开帷幕。

回到曾经是故乡的破败[shiganshina-district](/entities/shiganshina-district-希干希纳区)，[eren-yeager](/entities/eren-yeager-艾伦耶格尔)和调查兵团发现镇上异常地没有巨人的踪影。即使在外门被堵住后，他们仍然奇怪地没有遇到任何抵抗。任务进展顺利，直到[armin-arlert](/entities/armin-arlert-阿明阿诺德)对敌人的缺席高度怀疑，发现了异常……
