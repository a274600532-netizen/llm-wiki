### 产品发布/更新

1. **Grok 推出 Automations 功能：定时或邮件触发，自动执行任务并汇报结果**
   xAI 为 Grok 引入 Automations 功能，用户可描述一次任务，让 Grok 按计划（一次/每日/工作日/每周/每月/每年）或邮件触发自动运行。每次执行都是一次完整对话，结果保存至运行历史，支持邮件或应用内通知。

2. **ChatGPT 工作区支持文档、表格、幻灯片编辑**
   在 ChatGPT 工作区中可直接创建和编辑精美的文档、电子表格和幻灯片，进一步扩展 ChatGPT 的生产力场景。
   [来源](https://x.com/ChatGPTapp/status/2077826846373380535)

   [来源](https://mp.weixin.qq.com/s/WlGAeogkF_N5122nHA0TtQ)

   [来源](https://mp.weixin.qq.com/s/mQeBO0xC6Z1R0LqZX5TvNg)

5. **面壁智能开源企业AI数字员工平台 StaffDeck**

### 行业动态

7. **欧盟裁定 Google 必须向竞争对手开放 Android 和 Search，影响 Gemini 等 AI 服务**
   欧盟依据 DMA 裁定 Google 必须向竞争对手开放 Android 和 Google Search 的关键部分，包括允许第三方 AI 助手和搜索引擎获得更大访问权限，可能削弱 Google 对两大核心平台的控制。
   [来源](https://www.theverge.com/policy/966438/eu-google-android-ai-interoperability-search-data-dma)

8. **台积电上调2026年资本支出预测至600~640亿美元，A14制程进展顺利**
   台积电2026Q2财报说明会上调资本支出预测，董事长魏哲家表示预计2028年量产的A14制程（1.4nm）开发进展顺利，将成为比2nm更大、更持久的工艺节点。
   [来源](https://www.ithome.com/0/977/544.htm)

### 论文研究

9. **Moonshot AI 发布 PerceptionBench：多模态模型视觉感知能力诊断基准**
   从40多个现有基准中模型实际失败案例归纳出的视觉感知基准，包含10项原子感知能力和3000道验证题。所有测试模型准确率均未超过60%，表明模型更多是猜测而非真正感知。
   [来源](https://www.kimi.com/blog/perception-bench)

### 技巧与观点

10. **Anthropic 用 Claude Code 大规模迁移代码：Bun 百万行 Zig 转 Rust，两周完成**
    Anthropic 工程师用 Claude Code 在两周内将 Bun 的百万行 Zig 代码迁移至 Rust，100% 现有测试通过。消耗约 16.5 万美元 API 成本，但编译时间从八分钟降至两秒，二进制启动快 6 倍。
   [来源](https://claude.com/blog/ai-code-migration)
