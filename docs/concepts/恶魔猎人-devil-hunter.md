# 恶魔猎人 (Devil Hunter)

恶魔猎人是《电锯人》系列中的核心职业设定。恶魔猎人的职责是猎杀对人类构成威胁的恶魔，隶属于各种组织，其中最主要的是公安对魔特异课。

## 在《蕾塞篇》中的体现

在《Chainsaw Man Movie: Reze-hen》中，[denji](/entities/denji)在成为公安对魔特异课的恶魔猎人后，迅速适应了新的生活与责任。[makima](/entities/makima)作为精英恶魔猎人，是Denji的上司和操控者。恶魔猎人的身份是Denji与Makima之间权力关系的基础。

## 叙事功能

恶魔猎人这一设定为作品提供了核心冲突框架——人类与恶魔之间的对抗，以及恶魔猎人内部的人际关系与权力斗争。在《蕾塞篇》中，这一设定更多作为背景存在，焦点转向了角色之间的情感冲突。
