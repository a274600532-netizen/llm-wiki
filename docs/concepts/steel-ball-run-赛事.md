# Steel Ball Run 赛事

Steel Ball Run 是《Steel Ball Run: JoJo no Kimyou na Bouken》中的核心设定，一场横跨美国旧西部的跨国越野赛。

## 赛事设定

- **起点**: 加利福尼亚州圣地亚哥
- **赛程**: 超过6000公里
- **奖金**: 五千万美元
- **规则**: 参赛者可使用任何类型的交通工具
- **条件**: 每天需穿越上百公里的未知荒原

## 叙事功能

Steel Ball Run 赛事不仅是故事的背景舞台，更是推动情节发展的核心引擎。它汇聚了来自各地的参赛者，制造了竞争、冲突与合作的契机，同时穿越荒野的旅程本身也构成了角色成长与冒险的框架。
