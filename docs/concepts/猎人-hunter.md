# 猎人 (Hunter)

猎人是《[hunter-x-hunter-2011](/entities/hunter-x-hunter-2011)》世界观中的一种职业。猎人们致力于完成各种危险任务，从探索世界未知领域到寻找稀有物品和怪物。成为猎人的必经之路是通过[hunter-examination](/concepts/猎人考试-hunter-examination)。

猎人这一身份在作品中具有极高的地位和自由度，同时也伴随着巨大的风险。主角 [gon-freecss](/entities/gon-freecss-小杰富力士) 的父亲 [ging-freecss](/entities/ging-freecss-金富力士) 正是一名猎人，Gon 参加猎人考试的核心动机正是为了寻找他。
