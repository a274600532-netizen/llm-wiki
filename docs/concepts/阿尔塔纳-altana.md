# 阿尔塔纳 (Altana)

阿尔塔纳（Altana）是《银魂》世界观中的核心设定元素，指代地球的龙脉或能量流。

## 在最终章中的作用

在《[5-anime--5-39486--1xtfkzd](/entities/gintama-银魂)》中，[gintoki-sakata](/entities/坂田银时-gintoki-sakata)通过勘测地球的龙脉（Altana）来追踪[utsuro](/entities/虚-utsuro)的踪迹。Altana成为剧情驱动的关键机制，连接了银时的行动与虚的回归。[tendoshuu](/entities/天道众-tendoshuu)残党对永生的追求也与Altana密切相关。
