# 等价交换

等价交换（Equivalent Exchange）是炼金术的基本原则，其核心思想是"获得某物，必须付出同等的代价"。在《钢之炼金术师 FULLMETAL ALCHEMIST》中，这一原则既是炼金术的操作法则，也是贯穿全剧的哲学框架。

## 在《钢之炼金术师》中的角色

等价交换原则在作品中以多种形式呈现。艾尔利克兄弟试图通过人体炼成复活母亲，却付出了身体残缺的代价——这既是对等价交换的体现，也暗示了其局限性。作品通过这一原则探讨了牺牲、代价、生命价值等深刻主题。

## 与人体炼成的关系

[human-transmutation](/concepts/人体炼成)作为禁忌行为，直接挑战了等价交换原则的边界。人体炼成的失败表明，某些事物（如生命）的价值可能超出了物质交换的范畴。
