# 万事屋 (Yorozuya)

万事屋是《银魂》系列中主角三人组经营的万事代办店，由坂田银时、志村新八和神乐组成。他们接受任何委托——从找猫到打坏人——只要给钱什么都干。

## 组织特征

- **成员**: [gintoki-sakata](/entities/坂田银时-gintoki-sakata)（老板）、[shinpachi-shimura](/entities/志村新八-shinpachi-shimura)、[kagura](/entities/神乐-kagura)
- **地点**: 歌舞伎町
- **宗旨**: 什么工作都接，只要给钱

万事屋不仅是主角团体的名称，也代表了他们"什么工作都做"的生存方式，是作品的核心叙事框架。
