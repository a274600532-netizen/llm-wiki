# Denji

- **发布日期**: 2026年07月10日
Denji是《电锯人》系列的核心主角，也是《Chainsaw Man Movie: Reze-hen》的主要角色。他是一名[devil-hunter](/concepts/恶魔猎人-devil-hunter)，隶属于公安对魔特异课。

## 在《蕾塞篇》中的角色

在电影中，Denji已经迅速适应了作为恶魔猎人的新生活与责任。当最初的混乱平息后，精英恶魔猎人[makima](/entities/makima)约他出去约会，这次约会加深了他对Makima的感情，他发誓不会爱上其他人。然而，Denji很快遇到了看似无辜的咖啡馆女服务员[reze](/entities/reze)，由此陷入情感三角冲突。

## 关键关系

- **[makima](/entities/makima)** — 精英恶魔猎人，对Denji进行情感操控，通过约会加深Denji对她的依赖
- **[reze](/entities/reze)** — 看似无辜的咖啡馆女服务员，成为Denji情感三角冲突的另一端
