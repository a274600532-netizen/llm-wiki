# 阿尔冯斯·艾尔利克

阿尔冯斯·艾尔利克（Alphonse Elric）是《钢之炼金术师 FULLMETAL ALCHEMIST》的核心主角之一，爱德华·艾尔利克的弟弟。

## 背景

在兄弟二人尝试人体炼成复活母亲的事故中，阿尔冯斯失去了整个身体。他的哥哥爱德华牺牲了右臂，将阿尔冯斯的灵魂束缚在一副巨大的盔甲上，使其得以留在物质世界。阿尔冯斯因此以灵魂附身于盔甲的形态存在，与爱德华一同踏上寻找恢复身体方法的旅程。
