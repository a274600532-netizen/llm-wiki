# 天道众 (Tendoshuu)

天道众（Tendoshuu）是《银魂》系列中的核心反派组织，在最终章《[5-anime--5-39486--1xtfkzd](/entities/gintama-银魂)》中扮演关键角色。

## 在最终章中的角色

天道众入侵了O-Edo Central Terminal，成为故事起点事件。两年后，天道众残党仍在追寻永生，与[utsuro](/entities/虚-utsuro)相关联。[gintoki-sakata](/entities/坂田银时-gintoki-sakata)在勘测地球龙脉（[altana](/concepts/阿尔塔纳-altana)）时与他们遭遇。
