# Steins;Gate（命运石之门）

《Steins;Gate》是一部2011年播出的日本科幻悬疑动画，改编自5pb.与Nitroplus共同开发的视觉小说游戏。作品在MyAnimeList上获得9.07的极高评分，被广泛认为是科幻动画的经典之作。

## 剧情概要

故事围绕疯狂科学家[rintarou-okabe](/entities/rintarou-okabe)展开，他与朋友[mayuri-shiina](/entities/mayuri-shiina)和[itaru-hashida](/entities/itaru-hashida)共同创立了[future-gadget-laboratory](/entities/future-gadget-laboratory)。三人组发明了一台能将香蕉变成绿色粘液的微波炉，这一看似无用的装置却隐藏着操控时间的惊人能力。当冈部参加一场会议后，故事逐渐揭示出时间旅行背后的复杂真相与沉重代价。

## 主题与特色

- **时间旅行与因果律**：作品以时间操控为核心叙事驱动力，探讨改变过去带来的连锁反应与伦理困境。
- **悬疑与科学**：融合真实科学理论与悬疑叙事，构建出严谨而引人入胜的剧情。
- **角色关系**：主角团之间的互动与情感纽带是作品的情感核心。
