# 真选组 (Shinsengumi)

真选组（Shinsengumi）是《银魂》系列中的重要组织，在最终章《[5-anime--5-39486--1xtfkzd](/entities/gintama-银魂)》中与[yorozuya](/entities/万事屋-yorozuya)联手行动。

## 在最终章中的角色

在[tendoshuu](/entities/天道众-tendoshuu)入侵O-Edo Central Terminal两年后，重组后的真选组与万事屋联手对被占领的Central Terminal发起进攻。
