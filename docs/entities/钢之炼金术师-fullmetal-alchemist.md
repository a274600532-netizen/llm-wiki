# 钢之炼金术师 FULLMETAL ALCHEMIST

- **发布日期**: 2026年07月10日
《钢之炼金术师 FULLMETAL ALCHEMIST》（Fullmetal Alchemist: Brotherhood）是2009年播出的日本动画，改编自荒川弘的同名漫画。该作品在MyAnimeList上获得9.11的高分，被广泛认为是有史以来最伟大的动画之一。

## 剧情概要

故事围绕艾尔利克兄弟——爱德华和阿尔冯斯展开。兄弟俩因试图通过禁忌的人体炼成复活已故的母亲，而付出了惨痛的代价：阿尔冯斯失去了整个身体，爱德华失去了一条腿和一只手臂。为了将阿尔冯斯的灵魂固定在物质世界，爱德华将其束缚在一副巨大的盔甲中。兄弟二人从此踏上寻找贤者之石的旅程，以期恢复失去的身体。

## 核心主题

作品深入探讨了[equivalent-exchange](/concepts/等价交换)原则——炼金术的基本法则，也是贯穿全剧的哲学框架。禁忌的[human-transmutation](/concepts/人体炼成)作为故事的触发事件，引出了关于生命、牺牲、代价与人性的深刻思考。
