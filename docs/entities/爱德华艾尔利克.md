# 爱德华·艾尔利克

爱德华·艾尔利克（Edward Elric）是《钢之炼金术师 FULLMETAL ALCHEMIST》的核心主角之一，被称为"钢之炼金术师"（Fullmetal Alchemist）。

## 背景

爱德华与弟弟阿尔冯斯因试图通过禁忌的人体炼成复活已故的母亲，而付出了惨痛代价。爱德华在事故中失去了左腿，随后又牺牲了右臂，将阿尔冯斯的灵魂束缚在一副盔甲中。此后，他装上了机械铠（Automail）作为义肢。
