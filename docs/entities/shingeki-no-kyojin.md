# Shingeki no Kyojin (Attack on Titan)

《Shingeki no Kyojin》（进击的巨人）是谏山创创作的日本漫画及动画系列。该系列以人类与巨人之间的生存战争为核心，探讨了自由、牺牲与真相的主题。

## 系列构成

- **第一季** (2013)
- **第二季** (2017)
- **第三季 Part 1** (2018)
- **第三季 Part 2** (2019) — 参见 [5-anime--5-38524--hnv61b](/sources/5-anime--5-38524--hnv61b)
- **最终季** (2020–2023)

## 核心设定

- [wall-maria](/entities/wall-maria-玛利亚之墙) — 三道围墙中最外层的城墙，本季夺回行动的目标
- [shiganshina-district](/entities/shiganshina-district-希干希纳区) — 玛利亚之墙内的区域，艾伦的故乡
- [survey-corps](/entities/survey-corps-调查兵团) — 负责墙外调查与巨人作战的军事组织

## 关键角色

- [eren-yeager](/entities/eren-yeager-艾伦耶格尔) — 系列核心主角，拥有巨人化能力
- [armin-arlert](/entities/armin-arlert-阿明阿诺德) — 艾伦的挚友，以智谋著称
- [mikasa-ackerman](/entities/armin-arlert-阿明阿诺德) — 艾伦的养姐，顶尖战士
