## 剧情概要

故事设定在美国旧西部，一场名为"Steel Ball Run"的跨国越野赛成为整个故事的核心舞台。数千名参赛者从圣地亚哥出发，穿越超过六千公里的荒野，争夺五千万美元的巨额奖金。参赛者可以使用任何类型的交通工具，但必须忍受每天上百公里的严酷行程。
