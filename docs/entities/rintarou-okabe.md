# Rintarou Okabe（冈部伦太郎）

冈部伦太郎是《[steins-gate](/entities/steins-gate)》的核心主角，自称"疯狂科学家"，绰号"凤凰院凶真"。他对科学探索有着永不满足的渴望，性格中兼具中二病的夸张与深层的责任感。

## 角色定位

- **核心主角**：故事围绕冈部的视角展开，他的行动与选择驱动着整个时间旅行叙事。
- **疯狂科学家**：以夸张的言行和自称"疯狂科学家"的形象著称，但背后隐藏着敏锐的洞察力。
- **实验室创始人**：与[mayuri-shiina](/entities/mayuri-shiina)和[itaru-hashida](/entities/itaru-hashida)共同创立[future-gadget-laboratory](/entities/future-gadget-laboratory)。

## 关键关系

- **椎名真由理**：青梅竹马的朋友，冈部对她有着深厚的保护欲。
- **桥田至**：室友兼实验室伙伴，技术天才，是冈部的重要支持者。
