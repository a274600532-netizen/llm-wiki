# Gon Freecss (小杰·富力士)

Gon Freecss（小杰·富力士）是《[hunter-x-hunter-2011](/entities/hunter-x-hunter-2011)》的核心主角。故事开始时他 12 岁，参加[hunter-examination](/concepts/猎人考试-hunter-examination)的驱动力是寻找他的父亲 [ging-freecss](/entities/ging-freecss-金富力士)——一名猎人。Gon 相信，只要成为猎人，就能找到父亲的下落。

Gon 性格纯真、坚韧，拥有出色的直觉和身体素质。他的寻父之旅构成了作品早期叙事的主要驱动力。
