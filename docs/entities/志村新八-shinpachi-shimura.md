# 志村新八 (Shinpachi Shimura)

志村新八是《银魂》系列的核心角色之一，万事屋的成员。他是一名戴眼镜的少年，性格较为正经，经常担任吐槽役，对银时的懒散行为感到无奈。

## 角色特征

- **身份**: 万事屋成员，道场继承人
- **性格**: 认真、善良，偶尔热血
- **标志**: 眼镜、吐槽技能

在《Gintama°》中，新八继续与银时和神乐一起经营万事屋，同时抱怨自己迟迟拿不到工资。
