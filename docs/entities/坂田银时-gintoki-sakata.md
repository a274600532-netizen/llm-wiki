# 坂田银时 (Gintoki Sakata)

坂田银时是《银魂》系列的核心主角，万事屋的领导者。他是一名银发天然卷的武士，虽然表面上懒散、贪财、沉迷小钢珠，但在关键时刻展现出极强的正义感和战斗力。

## 角色特征

- **身份**: 万事屋老板，前攘夷志士
- **性格**: 懒散、贪财、毒舌，但重情重义
- **标志**: 银发天然卷、死鱼眼、爱喝草莓牛奶

在《Gintama°》中，银时继续带领万事屋团队接各种委托，同时面对来自天人势力的新挑战。
