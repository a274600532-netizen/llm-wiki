# Makima

Makima是《电锯人》系列中的精英[devil-hunter](/concepts/恶魔猎人-devil-hunter)，在《Chainsaw Man Movie: Reze-hen》中扮演关键推动者的角色。

## 在《蕾塞篇》中的角色

在电影中，Makima主动约[denji](/entities/denji)出去约会。这次约会强化了Denji对她的情感依赖，Denji甚至发誓不会爱上其他人。这一行为被解读为情感操控——Makima通过约会建立Denji对她的忠诚，为后续[reze](/entities/reze)的登场制造情感冲突。

## 叙事功能

Makima的约会行为是典型的"情感陷阱"叙事手法，旨在巩固她对Denji的控制。这一情节为电影的情感三角冲突奠定了基础。
