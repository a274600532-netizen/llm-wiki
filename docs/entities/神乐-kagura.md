# 神乐 (Kagura)

神乐是《银魂》系列的核心角色之一，万事屋的成员。她是一名来自夜兔族的少女，拥有超乎常人的怪力，性格天真活泼但有时暴力。

## 角色特征

- **身份**: 万事屋成员，夜兔族
- **性格**: 天真、贪吃、暴力倾向
- **标志**: 蓝色和服、雨伞（夜兔族怕阳光）、大胃王

在《Gintama°》中，神乐继续与银时和新八一起经营万事屋，同时抱怨自己迟迟拿不到工资。
