# Eren Yeager (艾伦·耶格尔)

艾伦·耶格尔是《Shingeki no Kyojin》（进击的巨人）系列的核心主角。他出生于[shiganshina-district](/entities/shiganshina-district-希干希纳区)，在目睹母亲被巨人吞噬后，立誓消灭所有巨人，并加入了[survey-corps](/entities/survey-corps-调查兵团)。

## 在第三季 Part 2 中的角色

在夺回[wall-maria](/entities/wall-maria-玛利亚之墙)的行动中，艾伦与调查兵团一同返回了曾经的家园希干希纳区。该季的故事围绕这一关键战役展开，艾伦的巨人化能力成为战斗的关键因素。
