# 虚 (Utsuro)

- **发布日期**: 2026年07月10日
虚（Utsuro）是《银魂》系列的核心反派，在最终章《[5-anime--5-39486--1xtfkzd](/entities/gintama-银魂)》中作为主要威胁登场。

## 在最终章中的角色

在[tendoshuu](/entities/天道众-tendoshuu)入侵O-Edo Central Terminal两年后，[gintoki-sakata](/entities/坂田银时-gintoki-sakata)预见到虚的回归，开始勘测地球的龙脉（[altana](/concepts/阿尔塔纳-altana)）以追踪虚的踪迹。虚与天道众残党相关联，后者仍在追寻永生。
