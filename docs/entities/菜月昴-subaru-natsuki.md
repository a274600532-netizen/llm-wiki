# 菜月昴 (Subaru Natsuki)

- **发布日期**: 2026年07月10日
菜月昴（Subaru Natsuki）是《Re:Zero kara Hajimeru Isekai Seikatsu》系列的主角。在第四季中，他面临着前所未有的挑战。

## 第四季中的角色定位

在经历了水门都市普利斯特拉的惨胜后，昴的同伴们——雷姆、库珥修、尤里乌斯——被"暴食"权能（[authority-of-gluttony](/concepts/暴食权能-authority-of-gluttony)）吞噬了存在。昴因此承担起寻找解救方法的责任。他的下一个目标是寻找全知的"贤者"莎缇拉（[sage-shaula](/entities/贤者莎缇拉-sage-shaula)），并前往贤者之塔（[pleiades-watchtower](/entities/贤者之塔-pleiades-watchtower)）。

## 核心动机

昴在本季的核心动机是拯救被剥夺存在的同伴。与以往的战斗不同，这次他面对的不是可以靠"死亡回归"简单重来的物理威胁，而是对同伴存在本身的抹除。这迫使他必须寻找新的解决路径，而非依赖重复的试错。

## 关键行动

- 在普利斯特拉战役中勉强取得胜利
- 得知"贤者"莎缇拉的存在
- 决定前往贤者之塔寻求解救同伴的方法
