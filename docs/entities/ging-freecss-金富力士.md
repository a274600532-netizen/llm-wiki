# Ging Freecss (金·富力士)

- **入库日期**: 2026-07-10
Ging Freecss（金·富力士）是《[hunter-x-hunter-2011](/entities/hunter-x-hunter-2011)》中主角 [gon-freecss](/entities/gon-freecss-小杰富力士) 的父亲，本人也是一名猎人。Gon 参加[hunter-examination](/concepts/猎人考试-hunter-examination)的核心动机正是为了寻找 Ging。Ging 在故事早期作为 Gon 的追寻目标而存在，其行踪和身份是推动剧情发展的关键要素。
