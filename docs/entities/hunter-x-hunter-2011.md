# Hunter x Hunter (2011)

《Hunter x Hunter (2011)》是一部由日本动画公司 Madhouse 制作的动画作品，于 2011 年 10 月至 2014 年 9 月播出，共 148 集，已完结。该作品改编自富坚义博的同名漫画，在 MyAnimeList 上获得 9.03 的高评分，被广泛认为是动作冒险奇幻类动画的经典之作。

## 剧情概要

故事围绕猎人这一职业展开。猎人们致力于完成各种危险任务，从探索世界未知领域到寻找稀有物品和怪物。成为猎人的必经之路是[hunter-examination](/concepts/猎人考试-hunter-examination)——一个高风险、高淘汰率的选拔过程，大多数申请者会因此致残甚至丧命。

挑战这场著名考试的参与者各有其理由。12 岁的 [gon-freecss](/entities/gon-freecss-小杰富力士) 参加猎人考试的驱动力是寻找他的父亲 [ging-freecss](/entities/ging-freecss-金富力士)——一名猎人。Gon 相信，只要成为猎人，就能找到父亲的下落。
