# 雷姆 (Rem)

雷姆（Rem）是《Re:Zero kara Hajimeru Isekai Seikatsu》系列的重要角色，鬼族的女仆。

## 第四季中的状态

在第四季开始时，雷姆已被"暴食"权能（[authority-of-gluttony](/concepts/暴食权能-authority-of-gluttony)）吞噬，陷入假死状态（suspended animation）。这是驱动本季剧情的关键事件之一。[subaru-natsuki](/entities/菜月昴-subaru-natsuki)寻找解救同伴的方法，很大程度上是为了拯救雷姆。

## 剧情意义

雷姆的困境代表了"暴食"权能造成的存在性剥夺——受害者并非死亡，而是其存在被部分或完全抹除。这种状态比死亡更令人绝望，因为它剥夺了受害者被拯救、被记住的可能性。昴的旅程因此具有了双重意义：不仅要恢复雷姆的意识，还要恢复她在世界中的存在。
