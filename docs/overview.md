# Overview

本Wiki当前聚焦于对十部日本动画作品的分析与知识组织：《葬送的芙莉莲》（Sousou no Frieren）、《Re:Zero kara Hajimeru Isekai Seikatsu 4th Season》、《钢之炼金术师 FULLMETAL ALCHEMIST》（Fullmetal Alchemist: Brotherhood）、《Steel Ball Run: JoJo no Kimyou na Bouken》（Steel Ball Run: JoJo's Bizarre Adventure）、《Steins;Gate》（命运石之门）、《Chainsaw Man Movie: Reze-hen》（电锯人 蕾塞篇）、《Shingeki no Kyojin Season 3 Part 2》（进击的巨人 第三季 Part 2）、《Gintama°》（银魂°）、《Gintama: The Very Final》（银魂：最终篇）以及《Hunter x Hunter (2011)》。

《葬送的芙莉莲》是一部2023年播出的日本动画（改编自同名漫画），在MyAnimeList上获得9.26的极高评分，以其叙事创新性和情感深度而备受赞誉。Wiki收录了作品实体页面（[sousou-no-frieren](/entities/sousou-no-frieren)）与核心角色页面（[frieren](/entities/frieren-芙莉莲)），并提炼出两个跨作品适用的核心概念：[long-lived-time-perception](/concepts/长寿种的时间感知-long-lived-time-perception)——探讨寿命差异带来的时间体验与情感鸿沟；以及[post-adventure-narrative](/concepts/冒险后的叙事-post-adventure-narrative)——一种以冒险结束为起点的叙事结构。

《Re:Zero》第四季是2026年播出的动画，在MyAnimeList上获得9.18的评分。本季故事始于水门都市普利斯特拉的惨胜之后，主角[subaru-natsuki](/entities/菜月昴-subaru-natsuki)的同伴被"暴食"权能（[authority-of-gluttony](/concepts/暴食权能-authority-of-gluttony)）吞噬了存在，他因此踏上寻找全知"贤者"莎缇拉（[sage-shaula](/entities/贤者莎缇拉-sage-shaula)）的旅程，前往贤者之塔（[pleiades-watchtower](/entities/贤者之塔-pleiades-watchtower)）。Wiki收录了作品实体页面（[re-zero-4th-season](/entities/re-zero-4th-season)）、核心角色页面（[subaru-natsuki](/entities/菜月昴-subaru-natsuki)、[rem](/entities/雷姆-rem)、[sage-shaula](/entities/贤者莎缇拉-sage-shaula)）、关键地点页面（[pleiades-watchtower](/entities/贤者之塔-pleiades-watchtower)）以及核心概念页面（[authority-of-gluttony](/concepts/暴食权能-authority-of-gluttony)）。

《钢之炼金术师 FULLMETAL ALCHEMIST》是2009年播出的动画，在MyAnimeList上获得9.11的评分，被广泛认为是有史以来最伟大的动画之一。故事围绕艾尔利克兄弟——[edward-elric](/entities/爱德华艾尔利克)和[alphonse-elric](/entities/阿尔冯斯艾尔利克)——因尝试禁忌的[human-transmutation](/concepts/人体炼成)而付出惨痛代价展开。Wiki收录了作品实体页面（[fullmetal-alchemist-brotherhood](/entities/钢之炼金术师-fullmetal-alchemist)）、核心角色页面（[edward-elric](/entities/爱德华艾尔利克)、[alphonse-elric](/entities/阿尔冯斯艾尔利克)）以及核心概念页面（[human-transmutation](/concepts/人体炼成)、[equivalent-exchange](/concepts/等价交换)）。

《Steel Ball Run: JoJo no Kimyou na Bouken》是《JoJo的奇妙冒险》系列的一部动画作品，于2026年播出，在MyAnimeList上获得9.1的高评分，目前状态为"Currently Airing"。故事设定在美国旧西部，一场名为"Steel Ball Run"的跨国越野赛成为核心舞台，参赛者从圣地亚哥出发穿越超过六千公里的荒野争夺巨额奖金。Wiki收录了作品实体页面（[steel-ball-run](/entities/steel-ball-run)）以及赛事概念页面（[steel-ball-run-race](/concepts/steel-ball-run-赛事)）。

《Steins;Gate》是2011年播出的科幻悬疑动画，在MyAnimeList上获得9.07的极高评分，被广泛认为是科幻动画的经典之作。故事围绕疯狂科学家[rintarou-okabe](/entities/rintarou-okabe)展开，他与朋友[mayuri-shiina](/entities/mayuri-shiina)和[itaru-hashida](/entities/itaru-hashida)共同创立了[future-gadget-laboratory](/entities/future-gadget-laboratory)，发明了一台能够操控时间的微波炉装置。Wiki收录了作品实体页面（[steins-gate](/entities/steins-gate)）、核心角色页面（[rintarou-okabe](/entities/rintarou-okabe)、[mayuri-shiina](/entities/mayuri-shiina)、[itaru-hashida](/entities/itaru-hashida)）以及组织设定页面（[future-gadget-laboratory](/entities/future-gadget-laboratory)）。

《Chainsaw Man Movie: Reze-hen》是2025年上映的动画电影，在MyAnimeList上获得9.06的高评分。故事围绕[denji](/entities/denji)在成为公安对魔特异课的[devil-hunter](/concepts/恶魔猎人-devil-hunter)后展开，精英恶魔猎人[makima](/entities/makima)通过约会对他进行情感操控，而看似无辜的咖啡馆女服务员[reze](/entities/reze)的出现则使Denji陷入情感三角冲突。Wiki收录了作品实体页面（[chainsaw-man-reze-movie](/entities/chainsaw-man-reze-movie)）、核心角色页面（[denji](/entities/denji)、[makima](/entities/makima)、[reze](/entities/reze)）以及核心概念页面（[devil-hunter](/concepts/恶魔猎人-devil-hunter)）。

《Shingeki no Kyojin Season 3 Part 2》（进击的巨人 第三季 Part 2）是2019年播出的动画，在MyAnimeList上获得9.05的高评分。本季讲述了[survey-corps](/entities/survey-corps-调查兵团)为夺回[wall-maria](/entities/wall-maria-玛利亚之墙)而发起的希干希纳区战役，[eren-yeager](/entities/eren-yeager-艾伦耶格尔)返回故乡却发现巨人异常缺席，这一悬念由[armin-arlert](/entities/armin-arlert-阿明阿诺德)提出，成为本季的关键叙事转折点。Wiki收录了来源页面（[5-anime--5-38524--hnv61b](/sources/5-anime--5-38524--hnv61b)）、系列实体页面（[shingeki-no-kyojin](/entities/shingeki-no-kyojin)）、核心角色页面（[eren-yeager](/entities/eren-yeager-艾伦耶格尔)、[armin-arlert](/entities/armin-arlert-阿明阿诺德)）、组织页面（[survey-corps](/entities/survey-corps-调查兵团)）、地点页面（[wall-maria](/entities/wall-maria-玛利亚之墙)、[shiganshina-district](/entities/shiganshina-district-希干希纳区)）以及概念页面（[wall-maria-retaking](/concepts/夺回玛利亚之墙)、[titan-absence](/concepts/巨人的缺席)）。

《Gintama°》（银魂°）是《银魂》系列的第四季动画，于2015年播出，共51集，在MyAnimeList上获得9.05的高评分。故事发生在一个被外星人（天人）征服、禁刀的架空江户时代，主角[gintoki-sakata](/entities/坂田银时-gintoki-sakata)、[shinpachi-shimura](/entities/志村新八-shinpachi-shimura)和[kagura](/entities/神乐-kagura)三人经营着万事屋（[yorozuya](/entities/万事屋-yorozuya)），靠接各种零活维生。该作品以其独特的[alternate-history-edo](/concepts/架空历史江户设定-alternate-history-edo)、无厘头喜剧与感人剧情并存的风格而闻名。

《Gintama: The Very Final》（银魂：最终篇）是《银魂》系列的最终章电影，于2021年上映，在MyAnimeList上获得9.05的高评分。故事发生在[tendoshuu](/entities/天道众-tendoshuu)入侵O-Edo Central Terminal两年后，[yorozuya](/entities/万事屋-yorozuya)成员已分道扬镳，[gintoki-sakata](/entities/坂田银时-gintoki-sakata)预见到[utsuro](/entities/虚-utsuro)的回归，开始勘测地球的龙脉（[altana](/concepts/阿尔塔纳-altana)）以追踪虚的踪迹。重组后的[shinsengumi](/entities/真选组-shinsengumi)与万事屋联手对被占领的Central Terminal发起进攻。Wiki收录了来源页面（[5-anime--5-39486--1xtfkzd](/sources/5-anime--5-39486--1xtfkzd)）、作品实体页面（[gintama](/entities/gintama-银魂)）、核心角色页面（[gintoki-sakata](/entities/坂田银时-gintoki-sakata)）、组织页面（[yorozuya](/entities/万事屋-yorozuya)、[shinsengumi](/entities/真选组-shinsengumi)、[tendoshuu](/entities/天道众-tendoshuu)）、反派页面（[utsuro](/entities/虚-utsuro)）以及概念页面（[altana](/concepts/阿尔塔纳-altana)）。

《Hunter x Hunter (2011)》是一部由Madhouse制作的动画作品，于2011年至2014年播出，共148集，在MyAnimeList上获得9.03的高评分。故事围绕猎人这一职业展开，猎人们致力于完成探索未知领域、寻找稀有物品等危险任务。成为猎人的必经之路是[hunter-examination](/concepts/猎人考试-hunter-examination)——一个高风险、高淘汰率的选拔过程。12岁的[gon-freecss](/entities/gon-freecss-小杰富力士)参加猎人考试的驱动力是寻找他的父亲[ging-freecss](/entities/ging-freecss-金富力士)——一名猎人。Wiki收录了作品实体页面（[hunter-x-hunter-2011](/entities/hunter-x-hunter-2011)）、核心角色页面（[gon-freecss](/entities/gon-freecss-小杰富力士)、[ging-freecss](/entities/ging-freecss-金富力士)）以及核心概念页面（[hunter-examination](/concepts/猎人考试-hunter-examination)、[hunter](/concepts/猎人-hunter)）。
